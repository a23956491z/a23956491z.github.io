Stock Backtesting

Language : Python

Stock Market : Taiwan

Visualize the indicator and stock price by mplfinance and matplotlib.

Calculate the buying signal and selling signal by specific conditions which is called strategy.

At first, we got stock price data from TEJ database.

It provides to download data in .CSV file.

After deleting useless information, we used pandas to import data and calculating the indicator we need, in this case, Stochastic Oscillator Indicator (KD).

We can now plot the price and our indicator. Following figures is basically the same in 2 forms.


Source code

Source code

Finally, trading with a specific strategy and printing the records. We can easily get return rate in multiple shares to evaluate the goodness of strategy on each share.

The strategy we used here :

  • Buy : When the KD indicator is golden cross and K > 70.
  • Sell : Trailing stop loss with 20%.
